Project for High Performance Computing class

Instructions:

1. open an interactive node on the discovery cluster.
2. compile the code with the command "nvcc -o main  main_CUDA.cu"
3. run the code with the command "./main" 

File Descriptions:

main_CUDA.cu - main CUDA code with CUDA parallelization implemented
main - executable 
F.txt - input file of atom form factors 
q.txt - input file of q angle values 
X.txt - input file of atom coordinates
